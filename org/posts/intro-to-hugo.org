* Create a new site
Create a new site with the name "quickstart".
#+BEGIN_SRC bash
  hugo new site quickstart
  cd quickstart
  git init
#+END_SRC
* Add an new theme
I added the theme.
#+BEGIN_SRC bash
  git submodule add https://github.com/vaga/hugo-theme-m10c.git themes/m10c
  echo 'theme = "m10c"' >> config.toml
#+END_SRC
* Add a post
#+BEGIN_SRC bash
  hugo new posts/my-first-post.md
#+END_SRC
